### 학습
- 파이썬 프레임워크에 식을 작성하면 쿠다 코드로 계산하고 모델을 생성 
- 가중치와 구조를 갖는다.모델파일 생성 
### 추론
- 가중치와 구조를 사용하여 추론 계산을 한다. 모델 파일을 사용한다.
### 순수 쿠다 코어
- 가중치와 구조는 메모리에 배열형태로 저장
- 학습 후 추론코드도 작성해야 한다.

### 아키텍쳐

- 추론 모델에 쿠다 api 있으면 x86 아키텍쳐이다.
- 학습 프레임워크에 쿠다 api 있으면 x86 아키텍쳐이다.
- 만약 학습은 x86에 하고 추론은 다른 아키텍쳐로 할수있다. 특정 api 콜이 없으면
