##### IPV4 Vs IPV6
주소 범위가 다르다.
공유기 의 등장으로 ip-v6가 필요가 없다.
ip를 공유한다.
유동 ip 개념

```c
//네트워크 주소체계
# define AF_INET //ip_4
# define AF_INET6 //ip_6

//소켓타입
# define SOCK_STREAM //tcp 전송 방식
# define SOCK_DGRAM //udp 전송방식

//protocol
//inprotocol udp
//inprotocol tcp
//0 자동으로 2번째 매개 변수기준으로 알아서
socket()
```
