su : root 계정으로 변경
.bashrc 사용자의 쉘개인 초기화 스크립트,시작 할때마다 자동 실행됨.

root계정의 .bashrc와 개인계정의 .bashrc는 다르다. 계정이 바뀌면 해당 초기화 스크립트 실행이됨

윈도우만 20년 쓴 나에게 갑자기 리눅스를 사용하라고?

```bash
KENEL=kenel8 # 변수 할당
echo $KENEL # 변수 호출
echo hello #그냥 호출
sudo passwd root # 패스워드 재설정
```

##### 색상 보이게하기
1. bashrc 파일:사용자의 bash 쉘 개인 초기화 스크립트,시작 할 때마다 자동 실행됨.
2. 색상이 나오는 사용자의 파일을 복사해서 root에도 적용
3. source 명령어 사용

##### alias 설정하기
1. .zshrc 파일 혹은 .bashrc에 들어간다.
``` bash
alias l="ls"# 첫번째 별명에 기존 명령어 할당하는 것이다.
```
##### 환경 변수 설정하기
```bash
export PATH="/custom/dir:$PATH"# 기존path경로에 앞에 저장
export PATH:"$PATH:/custom/dir"
#기존 경로 뒤에 저장한다.

#export를 하면 모든 하위프로세스도 그 변수의 값에 접근이 가능하다.
```
##### 파일 크기 확인 하기
```bash
ls -l #파잉이름 바이트 단위
```
