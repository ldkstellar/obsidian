[[리눅스 컴파일하기]]
디버깅을 하면서, 커널 코드를 함께 분석하면 다음 정보를 얻을 수 있음

-  분석 대상 코드가 동작하는 콜스택: 변수 리턴 주소 등등 저장
-  함수가 실행될 때 변경되는 자료구조:
- 함수가 실행되는 빈도와 실행시각
- 분석 대상 코드를 실행하는 프로세스

### Samba 설치

```bash
apt install samba # 삼바 설치

sudo smbpasswd -a username # 삼바 유저네임 등록하기
# etc/samba/smb.config 
```

### 2. smb.cofig 설정하기

``` config
[share]
   comment = superuser
   path = / 
   valid users = ojw
   browseable = yes
   create mask = 0777
   directory mask = 0777
   writable = yes
/

```
path는 접근 경로
