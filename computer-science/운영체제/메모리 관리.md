Memory <- 필요한 부분 올리기 <- 가상메모리 <- 실행파일 B

페이징
가상메모리를 고정된 크기인 페이지 단위로 나누는 기법이다.

구성:
- 가상 주소 공간: 페이지 단위로 쪼갬
- 물리 메모리(RAM):프레임 단위로 관리
- 페이지 프레임으로 매핑됨
- 이 매핑 정보를 페이지 테이블에 저장

핵심내용: 프로그램은 100MB지만, 그 중 실제로 사용하는 8MB만 RAM에 올려도 실행 가능!
남머지는 필요할대 불러오는 방식인 lazy loading으로 처리한다.

메모리에서 쫒아내기는 디스크 스왑이다.메모리의 연장 공간이다.
