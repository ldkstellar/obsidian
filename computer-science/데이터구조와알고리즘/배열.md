### Arrays in Compilers

```cpp
Addr(A[i]) // = L0 + (i-1) * w
			// base Addr ,index, size of Data type 

// row major 
Addr([i][j])// L0 + (i*n +j) * w

// column major
Addr[i][j] // L0 +(j*m +i) * w

// row major
Addr[i][j][k][l] // L0 + [i*d2*d3*d4 + j*d3*d4 + k*d4 + l] * w

// column major
Addr[i][j][k]][l] // L0 + [l *d1 *d2 *d3+ k *d1 *d2 + j*d1 +i] * w
```

### 1. Row-major Mapping

```cpp
Addr(A[1][2]) //= 200 +[i*4+2] *2 = 200+6*2

// base Addr,index ,size of data type

Addr(A[2][3]) // 200 +[2*4+3]*2

Addr[i][j][k][l] // L0 +[i*d2*d3*d4 + i2*d3*d4 + i3*d4 + i4] * w

```

### 2. Column-major Mapping

```cpp
Addr([1][2]) // 200 + [2 * 3 + 1]*2
Addr([1][3]) // 200 +[3 * 3 + 1]*2

Addr[i][j][k]][l] // L0 + [l *d1 *d2 *d3+ k *d1 *d2 + j*d1 +i] * w
```


### Conclusion
4 D -> 3 + 2 + 1
5 D -> 4+3+2+1
n D  -> n-1 + n-2 ...3+2+1 =  n(n-1)/2 = O(n^2)
