### Arrays in Compilers

```cpp
Addr(A[i]) // = L0 + (i-1) * w
			// base Addr ,index, size of Data type 

Addr([i][j])// L0 + (i* n+j)*w
```

### 1. Row-major Mapping

```cpp
Addr(A[1][2]) //= 200 +[i*4+2] *2 = 200+6*2

// base Addr,index ,size of data type
Addr(A[2][3]) // 200 +[2*4+3]*2
```

### 2. Column-major Mapping
```cpp
Addr([1][2]) // 200 + [2 * 3 + 1]*2
Addr([1][3]) // 200 +[]
```
