cpu는 결국 os랑 아키텍쳐랑 같이 개발해야겠네 컴파일러가 필수이다. 결국 소프트웨어 생태계가 필수적이다.

### arm64 파편화를 제조사들이 해결하는 방법 

- 칩에 맞춘 리눅스 커널과 드라이버를 최적화
- 런타임/VM 최적화
- 모델수를 줄여 리눅스커널과 드라이버를 최적화를 줄인다.
- UI와 앱 프레임워크 최적화

### **1. 업데이트 과정**

1. **구글 AOSP(Android Open Source Project) 업데이트**
    
    - 구글이 새로운 기능, 보안 패치, ART 최적화, API 개선 등을 포함한 **최신 안드로이드 코드** 제공
        
    
2. **제조사 맞춤 최적화**
    
    - HAL, 커널, 드라이버를 자사 SoC/센서/카메라/디스플레이에 맞게 튜닝
        
    - UI/프레임워크(예: 삼성 One UI) 최적화
        
    - 배터리, CPU/GPU 스케줄링, 메모리 관리 등 기기 성능 조정
        
    
3. **테스트 및 배포**
    
    - 안정성, 앱 호환성, 성능 테스트
        
    - 문제가 없으면 사용자에게 OTA(Over-The-Air) 업데이트 배포
