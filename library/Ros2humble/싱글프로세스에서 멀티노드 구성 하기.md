[[구성가능 한 node 작성하기]]

### 1. 이용가능한 컴퍼넌트들 찾기

```bash
ros2 component types
```

### 2. 게시자와 구독자를 이용한 ROS 서비스를 이용한 런타임 구성

1. 첫번째 shell에서 컴퍼넌트 컨테이너 실행

```bash
ros2 run rclpp_component component_container
```

2. 두번째 shell 을 열고  입증한다 컨테이너 실행중인지 ros2 커맨드라인드 툴을 통해 너는 컴퍼넌트의 이름을 봐야한다.
```bash
ros2 component list
/ComponentManager
```
