
### 1. 이용가능한 컴퍼넌트 찾기

``` bash
ros2 component types
```

### 2. 런타임 구성 Ros services를 퍼블리서와 구독자를 사용하여

첫번째 쉘에서, 컴퍼넌트 컨테이너를 실행시킨다.
```bash
ros2 run rclcpp_components component_container
```

두번째 쉘에서 컴퍼넌트 매니저가 잘실행되고있는지 확인한다.

```bash
ros2 component list
```

두번째 쉘에서 컴퍼넌트를 로드 시킨다.
명령어드 로드된 컴퍼넌트의 유니크 Id 뿐만 아니라 node name을 반환한다.

```bash
ros2 component load /ComponentManager composition composition::Talker
```

다음은 다른 노드를 로드하는 명령어이다.
```bash
ros2 component load /ComponentManager composition composition::Listener
```

 이 유틸리티 커멘드는 컨테이너의 상태를 검사 할수 있다
```bash
ros2 component list
```
