1. 새로운디렉토리를 만든다. 저장된 기록을 위한

```bash
mkdir bag_files
cd bag_files
```

2.  토픽을 고른다.
ro2 bag 단지 기록한다 발행한 메세지에 토픽에서

```bash
ros2 topic list
ros2 topic echo topic_name # 토픽애 오는 메세지를 출력
```

3. 싱글 토픽을 기록하기
	너가 선택한 토픽이 실행되기 전에
	새로운 터미널을 열고 bag_files 디렉토리로 이동한다.
	
``` bash
ros2 bag record <topic_name>
```
컨트롤+ c하면 기록을 중단한다.
