### 종료 옵션

- :q 그냥 종료
- :q! 강제종료
- :w 저장
- :w! 강제 저장
- :wq 저장하고 종료
- wq! 강제 저장하고 종료
- :e! 버퍼의 변경내용 버리고 마지막 저장 상태로 되돌리기

### 되돌리기
u
### 입력모드

 - esc 명령모드로
 - i (커서앞)
 - a(커서뒤)
 - I 대문자(문장 시작)
 - A(문장끝)

###  네비게이션 명령어 모드
- ctrl +u 위로 스크롤링
- ctrl +d 아래로 스크

- H :화면위  
- M :화면중간
- L :화면 끝

- gg :파일앞
- G :파일 끝
- 숫자 + G 특정라인으로 이동

- 0 문장앞으로 이동
- $ 문장 뒤로 이동

- w word 단어이동
- t
- b  backward 단어뒤로 이동

- ex: 3w 3단어씩 이동 8w 8단어씩 이동

	화살표랑 동일
	- h 좌
	- j 상
	- k 하
	- l 우

### 검색
- 앞으로 검색
- / 검색하고싶은 단어 그리고 enter
- ? 검색하고 싶은 단어 그리고 enter
- n 다음단어
### 지우기
 - d 글자 삭제
 - dd 줄 삭제 
 - dw 단어 삭제
### 검색지우기

- d/ + 단어 단어 전까지 삭제

### 선택하기

- v 선택하기 방향키이동
-  vaw 단어 선택
- ctrl +v 블럭선택 
- y 선택 복사
- d 선택잘라내기
- x 선택 삭제
- 불여넣기 p
- 입력 줄 추가  o 그리고 insert모드 변경 따라서 esc누른다.
