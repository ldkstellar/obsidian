```rust
let num =10; // 상수

let mut  num = 10; // 변수

const MAX:i32 =100; // 컴파일 시간에 값을 할당
```

| **특징**  | const     | let                |
| ------- | --------- | ------------------ |
| 변경 가능성  | 항상 불변     | mut 키워드 사용 시 변경 가능 |
| 값 결정 시점 | 컴파일 타임    | 런타임                |
| 사용 범위   | 프로그램 전체   | 선언된 블록 내           |
| 메모리 할당  | 최적화될 수 있음 | 스택 또는 힙            |
