##### 리스트 
```python
array = list((1,2,3))# 튜플로 list 로 변환
array2 = list(range(0, 10))
```

즉, 리스트는 생성자는 변경불가능한 이터러블 객체가 있어야 한다.range도 마찬가지다.이터러블객체이기때문에 iter메소드를 통해 이터레이터를 만들 수 있다.
