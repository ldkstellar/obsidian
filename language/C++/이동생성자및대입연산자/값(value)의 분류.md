
### lvalue란?

- 단일 식을 넘어 지속되는 개체
	- 주소가 있음
	- 이름이 있는 변수
	- const 변수
	- 배열 변수
	- 비트 필드 (bit-fields)
	-  공용 구조체(unions)
	- 클래스 멤버
	- 좌측 값 참조(&)로 반환하는 함수 호출
	- 문자열 리터럴
- 결국 지금까지 봐 온 많은 것들

### rvalue란?
- 사용되는 단일 식을 넘어 지속되지 않는 일시적인 값
	- 주소가 없는 개체
	- 리터럴(문자열 리터럴 제외)
