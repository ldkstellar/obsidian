
### lvalue란?

- 단일 식을 넘어 지속되는 개체
	- 주소가 있음
	- 이름이 있는 변수
	- const 변수
	- 배열 변수
	- 비트 필드 (bit-fields)
	-  공용 구조체(unions)
	- 클래스 멤버
	- 좌측 값 참조(&)로 반환하는 함수 호출
	- 문자열 리터럴
- 결국 지금까지 봐 온 많은 것들

### rvalue란?
- 사용되는 단일 식을 넘어 지속되지 않는 **일시적인 값**
	- 주소가 없는 개체
	- 리터럴(문자열 리터럴 제외)
	- 참조로 반환하지 않는 함수 호출
	- i++와 i-- (++i와 --i는 lvalue들)
	- 기본적으로 지원되든 산술식, 논리식,그리고 비교식
	- 열거형(enum)
	- 람다(lambda)

### rvalue 참조(&&)
- c++11이후에 새로 나온 연산자
- 기능상 &연산자와 비슷
- &연산자는 lvalue 참조에 사용
- && 연산자는 rvlaue참조에 사용

### std::move()
- rvalue 참조를 반환
- lvalue를 rvalue로 변환
