```cpp
#include <iostream>

#include <string>

#include <thread>

#include <vector>

  

int main(int argc, char const* argv[]) {

std::vector<int> list(100, 1);

int result = 0;

  

std::thread thread(

[](const std::vector<int>& v, int& result) {

for (auto item : v) {

result += item;

}

},

list,

std::ref(result)); 
thread.join();

std::cout << result << std::endl;

return 0;

}
```
- ref로 감싸는 이유는 기본적으로 스레드는 값을 복사한다. ref로 참조 객체를 만들어 스레드와 메인스레드간의 참조 연결해 준다.
