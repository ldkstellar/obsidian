
>즉 구조체에서 멤버변수들의 데이터가 몇 바이트 떨어져 있는지 확인 할 수 있는 것 이다.

```cpp
#include <cstddef>

#include <iostream>

struct Student {

const char *ID;

const char *Name;

int CurrentSemester;

};

  

int main(int argc, char const *argv[]) {

std::cout << "ID offset" << offsetof(Student, ID) << std::endl;

std::cout << "Name offset" << offsetof(Student, Name) << std::endl;

std::cout << "CurrentSemseter offset" << offsetof(Student, CurrentSemester)

<< std::endl;

return 0;

}
```

##### offsetof 매크로
```cpp
#define offsetof(type,member) 
```

- 매크로의 일종
- 특정멤버가 본인을 포함한 자료 구조의 시작점에서부터 몇 바이트 떨어져 있는지 알려줌
- 직렬화(serialize)나 역직렬화(desserialize)를 할 때 꽤나 유용


직렬화는 파일에 쓰는 것 역직렬화 데이터를 읽는 것
