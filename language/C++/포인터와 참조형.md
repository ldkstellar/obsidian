참조형 변수는 무조건 초기화를 해야된다.
참조형은 메모리 주소를 바꾸지 못한다.
참조형은 메모리의해제의 권한을 갖지 못한다.

|                      | **포인터 반환 (T*)**                  | **참조 반환 (T&)**              |
| -------------------- | -------------------------------- | --------------------------- |
| **메모리 관리**           | delete 필요할 수도 있음 필요 없음           | 필요없음                        |
| **nullptr 반환 가능 여부** | 가능(nullptr 반환 가능)                | 불가능                         |
| **로컬 변수 반환 가능 여부**   | 가능                               | 불가능 (Dangling Reference 위험) |
| **객체 복사 여부**         | 복사 안 함                           | 복사 안 함                      |
| **주로 사용되는 경우**       | 동적 할당된 객체, 소유권 이전이 필요한 경우 글로벌 변수 | 글로벌 변수, 멤버 변수 반환            |
결론: 참조형과 참조값은 엄면히 다르다.
