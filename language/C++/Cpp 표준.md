1. 왜 퍼블릭을 올려야 할까?  자바는 헤더파일과 소스코드가 분리가 안 되어 있다. 하지만 c++은 다르다. 헤더파일만 볼수 있기 때문에 퍼블릭을 위로가게 한다.

2.  입력 매개 변수는 참조로 하고 출력 매개 변수는 포인터로 한다.

3.  규칙이 엄격한 컴파일러도 있기 때문에 함수 아규먼트는 파라미터타입을 최대한 지키도록 한다. 캐스팅한다는 말이다. 느슨한 컴파일러는 자동으로 암시적으로 변환 한다. "-Werror" 이걸 사용하면 엄격한 타입을 사용가능하다.

## 구조체에 관한 코딩 표준
- c++에서는 구조체를 클래스처럼 쓸 수 있음
- 구조체는 c스타일로 쓰기
- struct는 순수하게 데이터뿐이어야 함.(Plain Old Data, POD)
- 멤버 변수 무조건 public
- 가상 함수 x
- 메모리 카피가 가능하다. 포인터는 주소를 가져온다.
