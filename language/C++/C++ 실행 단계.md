1. 전처리 (Preprocessing)
	- 전처리 지시문을 처리 define이나 include
	-  매크로 확장 , 주석제거, 파일 포함
2. 컴파일(Compilation)
	- 문법검사, 최적화,기계어 변환 준비 작업이 이루어진다.
	- 어셈블리 코드가 기계어로 변환된다.
3. 어셈블 
	-  어셈블리 코드가 바이너리 코드로 변환
4. 링킹(Linking)
	-  여러 오브젝트 파일과 라이브러리를 연결하여 실행 가능한 바이너리 파일을 생성한다.
	
- 오브젝트 파일은 소스코드와 기계어로 변환한 중간 결과물 링커에서 여러 오브젝트 파일들을 합쳐서 실행파일은 만든다.
