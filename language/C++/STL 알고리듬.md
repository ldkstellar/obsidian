#### sort 알고리즘 
 - 비교 함수 형태: bool compare()

 반환 값의 의미 차이
 - true 값이면 a는 b보다 앞에 있어야 한다.
 - 앞에 있으면 그대로 아니면 위치를 바꾼다.
 - false 값이면 a는 b보다 앞에 있으면 안된다.
 - 앞에있으면 자리바꾸고 아니면 그대로
