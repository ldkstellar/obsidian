### 1.클래스에서 객체 복사

구조체는 구조체객체 그게 값이다. copy 받으면 그 값을 다 복사에서 주는 것이다. 깊은 복사 new를 사용한 heap 메모리를  복사를 하는 것은 얕은 복사이다.
클래스와 구조체의 멤버변수는 offset 기반으로 작동한다.

```cpp
class A {
    int x;     // offset +0
    char y;    // offset +4 (정렬 padding 가능)
};

```

```asm
// 가상 코드
mov eax, [this + 0]   ; x
mov bl,  [this + 4]   ; y
```
