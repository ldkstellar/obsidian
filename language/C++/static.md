- 범위(scope)의 제한을 받는 전역변수
- 어떤 범위?
	- 파일속
	- 네임스페이스 속
	- 클래 속
	- 함수 속

extern 키워드
- 다른 파일의 전역 변수에 접근을 가능케 해줌

static변수 파일안에서 내함수 밖에서 못 사용하게
함수속 정적변수 짱 박혀있다. 들어올때마다 초기화 돼지 않는다. scope로 제한된 전역 변수

##### 정적 멤버 변수 베스트 프랙티스
- 함수안에서 정적 변수를 넣지 말 것
	- 클래스 안에 넣기
- 전역변수 대신 정적 멤버변수를 쓸것
	- 범위(scope)를 제한하기 위해
- C스타일의 정적변수를 쓸 이유가 이제 없음

##### 정적 멤버 함수
-  논리적 범위(scope)에 제한 된 전역 함수
- 해당 클래스의 정적 멤버에만 접근 가능
- 개체가 없이도 정적 함수를 호출가능

```cpp
Math::Square(10);
```
